# Contributing to {{PLUGIN_NAME}}

This project is a Claude Code plugin — its primary source of truth is a set of Markdown
instruction files consumed directly by Claude, not compiled code.

---

## Ways to Contribute

- **Bug reports** — something behaves differently from what the docs say
- **Documentation fixes** — typos, unclear phrasing, outdated examples
- **Feature proposals** — open an issue before writing code; discuss the design first

---

## Bug Reports

Use the [bug report template](.github/ISSUE_TEMPLATE/bug_report.md).

Include the exact command you ran, your Claude Code version, and what happened vs. what you expected.

---

## Pull Requests

1. **Open an issue first** for anything beyond a typo fix.
2. **One concern per PR.** Do not bundle unrelated changes.
3. **Keep instruction files precise.** Claude executes these files literally. Vague wording causes behavioural drift.
